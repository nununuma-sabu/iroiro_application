# Pink Noise Generator

Pythonを用いて**ピンクノイズ（1/f ノイズ）**のWAVファイルを生成するコマンドラインツールです。
ホワイトノイズに比べて低音成分が強く、雨音や滝の音に近い「柔らかい」響きが特徴です。集中作業用BGM、リラックス、またはオーディオ機器のサウンドチェックにご利用ください。

## 特徴

* **1/f ゆらぎ**: FFT（高速フーリエ変換）を用いてホワイトノイズを周波数領域で加工し、正確なピンクノイズ特性（パワーが周波数に反比例）を実現しています。
* **WAV出力**: 非圧縮のWAV形式で出力するため、高音質です。
* **カスタマイズ可能**: 再生時間、音量、サンプリングレート、チャンネル数を引数で指定できます。

## 必要要件

* Python 3.6 以上
* NumPy

## インストール

1.  このリポジトリをクローンまたはダウンロードします。
2.  依存ライブラリ（NumPy）をインストールします。

```bash
pip install numpy

## 基本的な実行

引数を指定せずに実行すると、デフォルト設定（10分間、音量0.3、ステレオ）で `pink_noise.wav` が生成されます。

```bash
python pink_noise_generator.py
