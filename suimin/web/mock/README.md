# 🔊 Pink Noise Streaming Generator Mockup

## 🚀 プロジェクト概要

このプロジェクトは、**ピンクノイズ（1/f ノイズ）**を生成・ストリーミング再生するためのWebアプリケーションのモックアップ（初期プロトタイプ）です。

周波数に反比例してパワーが減衰するピンクノイズは、リラックスや集中を促すBGMとして人気があります。本プロジェクトは、フロントエンド学習とシステム構成検証を兼ねて開発を進めます。

## 🎯 目的（このリポジトリの役割）

1.  **Next.js/React学習**：モダンなフロントエンド技術を用いたコンポーネント設計と状態管理の実践。
2.  **システム構成の検証**：フロントエンド（Next.js）とバックエンド（Python/API）の連携およびノイズストリーミングの実現可能性検証。
3.  **デザインとUXの改善**：初期モックアップの機能性を保ちつつ、よりモダンで使いやすいUI/UXを実現する。

## 🛠️ 技術スタック

| 分野 | 技術 | 備考 |
| :--- | :--- | :--- |
| **フロントエンド** | **Next.js** (React) | 開発の中心。コンポーネントとルーティングを担当。 |
| **スタイル** | Tailwind CSS / CSS-in-JS (予定) | 今後モダンなデザインを実現するために導入予定。 |
| **ノイズ生成** | Python (Flask/FastAPI 連携予定) | 既存の `pink_noise_generator.py` をAPI経由で利用。 |
